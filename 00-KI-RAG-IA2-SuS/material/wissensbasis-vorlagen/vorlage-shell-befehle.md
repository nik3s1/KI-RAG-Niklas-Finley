# Häufige Linux-Befehle

## ls

Zeigt Dateien und Verzeichnisse an.

## cd

Wechselt das Verzeichnis.

## pwd

Zeigt das aktuelle Verzeichnis an.

## grep

Durchsucht Text nach bestimmten Mustern.

Beispiel:

grep "Fehler" logfile.txt

## Symbolic Link

Ein Verweis auf eine andere Datei.

Erstellen:

ln -s ziel linkname

## Hard Link

Verweist direkt auf dieselben Dateidaten wie die Originaldatei.
