# Wissensbasis-Vorlagen

Für jede Domäne gibt es hier eine **Start-Vorlage**. Sie ist bewusst
unvollständig – eure Aufgabe ist es, sie mit korrekten, gut strukturierten
Inhalten zu füllen und zu erweitern (siehe Arbeitsblatt 3).

| Datei | Domäne |
|-------|--------|
| `vorlage-cisco.md` | Cisco-Netzwerktechnik |
| `vorlage-csharp.md` | C# / Programmierung |
| `vorlage-datenbanken.md` | SQL / Datenbanken |
| `vorlage-linux.md` | Linux / Shell |
| `system-prompt-vorlage.md` | System-Prompt für euer Modell |

## So nutzt ihr die Vorlagen
1. Vorlage eurer Domäne kopieren.
2. Inhalte ergänzen/korrigieren – nutzt euer eigenes Unterrichtswissen.
3. Gute Struktur beachten: klare Überschriften, kurze Absätze, Code in Blöcken.
4. In Open WebUI als Knowledge hochladen.
